
### 🌟 Reverse_Star_Tower Program (Java)

This program prints a **reverse star pattern** based on the number of rows entered by the user:

1. The program asks the user: **“Put a number of rows:”**  
2. The user enters a number (e.g., 5).  
3. If the number is greater than 21, the program exits immediately.  
4. Otherwise, it prints stars (`*`) starting from the given number down to 1, forming a reverse triangle.  


### Example Output

If the user enters `5`, the output will be:

```
*****
****
***
**
*
```
